### ====DESCRIPTION===
### EduTrack is a web app, developed as a standalone application and using MVC as its 
### architecture, designed to improve communication between students and teachers by providing an 
### all-in-one solution that allows teachers to upload lectures, track attendance, assign 
### grades, and communicate with students through a built-in chat system. It will allow 
### students to view materials, monitor their performance, and communicate efficiently.
